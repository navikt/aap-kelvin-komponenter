# AAP Kelvin Komponenter

Felles-bibliotek for apper for AAP.

# Komme i gang

For oppdatert oppskrift for å kjøre koden, se stegene i Github Actions.

## Bygge dokumentasjon

```
./gradlew dokkaHtmlMultiModule
```

Åpne `index.html` i `build/dokka/htmlMultiModule`.
